\textbf{Question}:\\
The coordinates of the point P dividing the line segment joining the point A(1,3) and B(4,6) in the ratio 2:1 are
\hfill (10, 2012)
\\
\textbf{Solution: }
\begin{table}[h!]    
  \centering
  \input{tables/table.tex}
  \caption{Variables Used}
  \label{tab1-1.5-29}
\end{table}\\
If $P$ divides $AB$ in the ratio $k:1$,


\begin{align} 
P=\frac{kB+A}{k+1} 
\end{align}\\
Putting the values of A,B we get
\begin{align}
	P = \frac{\myvec{
4k+1 \\
6k+3
}}{k+1} 
\end{align}\\
here, $k = 2$, so putting the $k$ we get
\begin{align}
P = 
\myvec{
3\\
5
}
\end{align}
The coordinates of the required point $P$ are
\begin{align}
\myvec{
3\\
5
}
\end{align}
